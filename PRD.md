# PRD - Simple Dark Mode Website

## Overview

Build a single-page, production-ready marketing website with a dark mode theme.

## Requirements

- Single page site (no auth, no database).
- Dark mode look and feel.
- Responsive layout for mobile and desktop.
- Clean, modern UI with clear sections.

## Tasks

- [ ] Create a simple landing page with a hero section.
- [ ] Add a features section (3–5 items).
- [ ] Add a testimonials section (2–3 quotes).
- [ ] Add a final call-to-action section.
- [ ] Add a footer with basic links.

## Acceptance Criteria

- Page renders without errors.
- Layout looks good on mobile and desktop.
- Dark mode styling is consistent across sections.
